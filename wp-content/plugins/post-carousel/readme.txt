== Description ==
Post Carousel is a responsive carousel slider plugin for WordPress that allows you to show the list of posts in a nice sliding manner. It has plenty of user-friendly options to configure the carousel easily.




== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use Post Carousel in your page, widget or anywhere you like. How to create Post Carousel? lets see.


1. Upload the folder post-carousel to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
3. Call shortcode to your WordPress editor directly, by using
`
[post-carousel id="1" ]
`
Call shortcode to your php template file using
`
<?php  echo do_shortcode('[post-carousel id="1"]'); ?>
`


Now save your page, then visit the newly created page & see something awesome. :)
